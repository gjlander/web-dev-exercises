# WBS Spinner Galore

You need to fetch some data!

## Clone into directory

```sh
git clone git@github.com:WebDev-WBSCodingSchool/wbs-spinner-galore.git
```

## Install

```sh
cd wbs-spinner-galore && npm ci
```

## Start Dev Server

```sh
npm run dev
```
