Contact form
============

**Objective:**

Create a contact form.

**Instructions:**

*   Create a contact form with:  
    *   Name field
    *   Email field
    *   Telephone field
    *   Message field
*   Control the form with a single piece of state
*   On form submission:
    *   Prevent the default behaviour
    *   Validate that all fields have data
    *   Output form data to console.
